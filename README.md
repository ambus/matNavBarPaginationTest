# MatNavBarPaginationTest

Demo how to use [matNavBarPagination](https://www.npmjs.com/package/mat-nav-bar-pagination)


![](animation.gif)


## How to use

Install all dependency:
```bash
$  npm install
```
or yarn
```bash 
$  yarn install
```

Run with angular CLI command: `ng serve`.
